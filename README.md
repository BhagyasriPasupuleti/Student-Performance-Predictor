Student Performance Predictor

Overview

This project predicts a student's marks based on study hours and attendance using Machine Learning.

Technologies Used

- Python
- Pandas
- Scikit-Learn

Features

- Predict marks based on study hours
- Predict marks based on attendance
- Simple and beginner-friendly ML model

How to Run

1. Install Python
2. Install required libraries:
   pip install pandas scikit-learn
3. Run:
   python app.py

Sample Input

Study Hours: 8

Attendance: 90

Sample Output

Predicted Marks: 84.0
